デモ、デモパーティ、デモシーンについて

デモ（メガデモ、イントロ）
デモ は、 映像と音をリアルタイムに生成するプログラムで、 リアルタイムでどれだけ凄いものが動かせるか？を競う目的で作成されます。 デモにはサブカテゴリとして、 フロッピーディスク 1 枚という制約の中で作成される メガデモ 、64 キロバイトや 4 キロバイトの実行ファイル（外部データを参照しない）のみで構成される イントロ 等々があります。

デモパーティとデモコンポ
デモは、 デモパーティ と呼ばれるイベントで開催されるコンペティション（コンテスト）のエントリ作品として発表されます。 このコンペティションは デモコンポ と呼ばれ、大勢の観客が見守るなかエントリ作品が巨大なスクリーン上で実行されます。 そして、その場の投票によりエントリ作品の評価が行われます。

デモシーン
デモシーン とは、このようなデモやデモパーティおよびその周辺の歴史的背景も含めた文化の総称です。 デモシーンに関わる人たちのことは デモシーナ と呼ばれています。

デモシーンの歴史

デモパーティ発生の経緯
デモシーンは、 インターネットが出現する以前の 1980 年代、 北欧の若者たちの間で流行ったゲームソフトのクラッキング （コピープロテクトを解除し、違法にコピーできる状態にすること） が起源です。 彼らは、 クラッキングしたソフトにクレジットとして クラックトロ と呼ばれる独自の起動デモを挿入して配布していました。

クラックトロは作成者の技術力を誇示するため、音を伴った映像作品になっていました。 グラフィティ・アートが描きにくいところに描くほど評価が高くなるのと同様、 クラックトロは高度なプロテクトが掛けられたソフトをクラッキングし、 そのディスクの狭い空き領域に高度なデモを挿入することが高い評価につながります。 やがてこれが独立して、デモに発展していきます。

そしてヨーロッパの各地でコンピュータ愛好家が集まるパーティが開催されました。 パーティではデモコンポが行われた他、 各自が持ち寄った PC を LAN 接続し、ファイル交換パーティやゲーム大会が開催されました。 1990 年代初頭には、当時流行ったレイヴ・カルチャーも取り込まれました。 次第にファイル交換やゲームだけが目的の人達とは住み分けが進み、 デモに特化した「デモパーティ」に変化しました。

コンピュータ・アート・アンダーグラウンドが融合した独特な文化が成立
現在のデモシーンでは、クラッキングや違法コピーのような行為は行われておらず、 純粋にデモとそれに関連するコミュニティとして機能しています。 しかしそのルーツとして存在したアンダーグラウンド感は残り、 コンピュータ・アート・アンダーグラウンドが融合した独特な文化が成立しました。

「デモシーンとは何か？」について説明される際、必ずと言ってよいほどクラッキング・クラックトロの歴史についても言及されます。 それらの歴史は「デモシーンが何をもってアンダーグラウンドなのか？」の根拠になっています。 アンダーグラウンド要素はデモシーンにとって重要なものであり、切り離して考えることは難しいです。

日本のデモシーン

1990 年代後半、国内でもデモシーンの存在が広く知られるようになる
1990 年代後半、国内の雑誌にメガデモ紹介記事が盛んに掲載され、 デモシーンの存在が広く知られるようになりました。 当時の雑誌は、デモのことを「メガデモ」と称して紹介した経緯があり、 日本ではデモの総称としてメガデモという呼称が浸透することになります。 日本では「デモ」という言葉は他の意味（ゲーム内のカットシーン、販促 PR 用、プラカードを掲げて街を練り歩く等々）で使われることが多く、 混同を避ける理由からも便宜上「メガデモ」と呼ばれることが多いです。

日本初のデモコンテスト HOLICS'96
1996 年に日本国内で HOLICS'96 というデモコンポが開催されました。 web archive 上でその痕跡が確認できます。
web archive : DemoholicML 主催 HOLICS'96
HOLICS'96 ではオンラインでエントリ作品が募集されました。 それらを発表するためのパーティに相当するイベントが別途開催されたか、 どのように開催されたか等々については、 記録等がなく詳細が不明です。

２ちゃんねる発祥のデモパーティ 2chparty（2001 年～ 2009 年）
2001 年から、２ちゃんねる発祥のオンラインデモパーティである 2chparty が、不定期で開催されています。 これまでに、2001 年、2005 年、2006 年、2009 年の計 4 回が開催されました。

日本初の本格的なデモパーティ Tokyo Demo Fest（2011 年～）
Tokyo Demo Fest は、日本初の本格的なデモパーティです。 このパーティは海外とほぼ同じ形式で定期開催されています。 海外デモシーンからも主要なデモパーティの一つとして認知されていて、 国内外から多くの参加者が訪れます。

デモシーンの中心的な Web サイト

アーカイブサイト
デモパーティでリリースされた作品は、アーカイブサイトに掲載・保管されます。 アーカイブサイト上では、掲載された作品に対してコメントが付けられ再評価が行われます。 手加減なしのとても辛辣なコメントが付く傾向があります。

アーカイブサイトには以下のようなものがあります。
Pouet https://www.pouet.net/
2020 年現在、最も活発なサイト。 どう発音するのかが謎ですが、youtube のセミナー動画で「プエット」と発音されている様子が確認されています。

Demozoo https://demozoo.org/
こちらも 2020 年現在最も活発なサイトの一つです。

Scene org https://scene.org/
デモパーティでリリースされたファイルが保管されています。

アーカイブサイトの役割
デモシーンの世界では、 ある技術的な課題を最初にクリアした作品や 流行のスタイルを最初に持ち込んだ作品には、特に高い評価が与えられます。 このような作品に対して「一番乗り」「元祖」であるというお墨付きを与える根拠として、 それ以前の作品をいつでも参照できる状態にする必要があります。 したがって、デモシーンが過去作をアーカイブすることは、 単に歴史を記録して懐かしむことだけではなく、 新しいデモの評価体制を万全の状態に保つという目的もあります。
